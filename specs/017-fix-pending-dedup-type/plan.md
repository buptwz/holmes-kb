## Summary

Fix two issues from v15 verification report:
- **D-5**: `_find_entry_by_hash()` does not scan `contributions/pending/` — extend the scan to cover pending entries so duplicate imports are detected and skipped.
- **E-2**: `--type` CLI flag is received by `import_cmd` but never passed to `ImportAgentRunner` or `ThreePhaseImportPipeline` — thread `force_type` through the call chain and enforce it deterministically post-extraction.

Both fixes are surgical (1–4 lines each at the enforcement sites), with no schema changes and full backward compatibility.

### D-5: Extend `_find_entry_by_hash()` to scan pending directory

**File**: `kb/holmes/kb/agent/tools.py`

**Current code** (lines 61–73):
```python
def _find_entry_by_hash(kb_root: Path, source_hash: str) -> tuple[str | None, str | None]:
    """Return (entry_id, file_path) for the first entry matching source_hash."""
    for entry in list_entries(kb_root):
        file_path = Path(entry.file_path)
        if not file_path.exists():
            continue
        try:
            post = fm.load(str(file_path))
            if str(post.metadata.get("source_hash", "")) == source_hash:
                return entry.id, entry.file_path
        except Exception:
            pass
    return None, None
```

**Fixed code**:
```python
def _find_entry_by_hash(kb_root: Path, source_hash: str) -> tuple[str | None, str | None]:
    """Return (entry_id, file_path) for the first entry matching source_hash.

    Scans approved KB entries first, then contributions/pending/.
    """
    # Scan approved KB entries (existing behaviour).
    for entry in list_entries(kb_root):
        file_path = Path(entry.file_path)
        if not file_path.exists():
            continue
        try:
            post = fm.load(str(file_path))
            if str(post.metadata.get("source_hash", "")) == source_hash:
                return entry.id, entry.file_path
        except Exception:  # noqa: BLE001
            pass

    # D-5: Also scan contributions/pending/ so reimports are skipped.
    pending_dir = kb_root / PENDING_DIR
    if pending_dir.exists():
        for pending_file in sorted(pending_dir.glob("*.md")):
            try:
                post = fm.load(str(pending_file))
                if str(post.metadata.get("source_hash", "")) == source_hash:
                    pending_id = pending_file.stem
                    return pending_id, str(pending_file)
            except Exception:  # noqa: BLE001
                pass

    return None, None
```

**Import needed**: `PENDING_DIR` — already transitively available via `from holmes.kb.pending import write_pending` (same module). Add explicit import: `from holmes.kb.pending import PENDING_DIR, write_pending`.
